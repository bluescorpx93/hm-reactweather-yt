### React Tryout

Followed a tutorial to try React & OpenWeather.

#### To Test This

- Setup API_KEY.js
```sh
$ cd hm-react-weather-yt
$ touch src/API_KEY.js
```

Update API_KEY.js accordingly:
```
const key = <YOUR_OPENWEATHER_KEY>
export default key
```

- Install Package Dependencies
```sh
$ cd hm-react-weather-yt
$ yarn install 
#or with npm
$ npm install 
```

- Run Prod Mode
```sh
$ cd hm-react-weather-yt
$ yarn install
$ yarn start-app
```

- Run Dev Mode
```sh
$ cd hm-react-weather-yt
$ yarn install
$ yarn start
```
